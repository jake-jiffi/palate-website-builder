# Design mode taxonomy

Three modes, locked across the Jiffi ecosystem. Each is a token override set plus a set of component behaviours. A client project gets exactly one mode; the reference library tags each site by the mode it best exemplifies.

## Clean Corporate
Inter or similar neutral sans, restrained palette, conservative spacing, subtle shadows. Motion is functional (fades, gentle reveals), never decorative. Reads as trustworthy and modern. The safe default for B2B.
Token character: neutral greys, one accent, 1.2 type scale, shadow-sm to shadow.

## Motion-Heavy Creative
Display typeface with tight tracking, often a darker palette, generous spacing, GSAP-driven hero and scroll storytelling. Motion is a feature, not decoration. Reads as confident and premium.
Token character: high contrast, expressive accent, larger type scale, deliberate negative space, scroll-linked animation.

## Minimal Portfolio
Editorial serif or refined sans, abundant whitespace, a single accent colour, minimal motion. The content (work, words, imagery) carries the page. Reads as considered and quiet.
Token character: near-monochrome, one restrained accent, large type for headings, very generous spacing, motion limited to page transitions.

## Choosing a mode

The brief usually signals it. Conservative industry or risk-averse buyer leans Clean Corporate. Premium positioning or a brand that wants to feel distinctive leans Motion-Heavy Creative. Work-led or word-led brands lean Minimal Portfolio. When unsure, default Clean Corporate and offer to shift.
