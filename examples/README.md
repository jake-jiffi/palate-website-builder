# Examples

See references/examples.md for invocation patterns. A full worked example (the jiffi.com.au pilot) will be added here after Pilot 1 validates the package gate.

## The pilot gate
Pilot 1 (jiffi.com.au) answers one question: does the private brand package install in CI and deploy cleanly to Cloudflare Workers? If yes, the package path is the default. If no, --vendor-brand becomes default. This is the single most important thing the first build proves.
